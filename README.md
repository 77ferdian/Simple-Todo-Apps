# Todo List App

Aplikasi web sederhana untuk mengelola daftar tugas harian (Todo List) yang dibuat menggunakan HTML, CSS, dan JavaScript vanilla (tanpa framework). Aplikasi ini dirancang dengan antarmuka yang bersih dan interaktif untuk memudahkan pengguna mencatat hal-hal yang perlu dilakukan.

## 🚀 Fitur Utama

- **Tambah Tugas Baru:** Masukkan judul tugas beserta target tanggal penyelesaiannya.
- **Daftar Tugas Aktif:** Menampilkan daftar tugas yang belum selesai dilakukan.
- **Tandai Selesai:** Mengubah status tugas dari belum selesai menjadi selesai dengan satu tombol klik.
- **Kembalikan Status (Undo):** Mengembalikan tugas yang sudah selesai kembali ke daftar tugas aktif jika diperlukan.
- **Hapus Tugas:** Menghapus tugas yang sudah selesai dari daftar secara permanen.

## 🛠️ Teknologi yang Digunakan

- **HTML5:** Untuk struktur halaman web.
- **CSS3:** Untuk styling tampilan halaman (responsif dan rapi).
- **JavaScript (ES6):** Untuk logika pemrosesan data todo list secara dinamis menggunakan manipulasi DOM.

## 📂 Struktur Folder

```text
├── assets/          # Folder untuk aset gambar atau ikon (jika ada)
├── css/
│   └── style.css    # Berisi styling untuk aplikasi
├── js/
│   └── script.js    # Logika fungsionalitas Todo List (DOM & event handler)
├── index.html       # Halaman utama aplikasi web
└── README.md        # Dokumentasi proyek ini
```

## 💻 Cara Menjalankan Proyek secara Lokal

1. Klon repositori ini atau unduh file zip proyek.
2. Buka folder proyek di komputer Anda.
3. Klik dua kali pada file `index.html` untuk membukanya langsung di browser favorit Anda, atau jalankan menggunakan ekstensi seperti **Live Server** di VS Code.
